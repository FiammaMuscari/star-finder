# ⭐ Star Finder

<img width="1333" height="670" alt="image" src="https://github.com/user-attachments/assets/4ac5d63e-8181-4258-9e4b-2aa443fa619d" />


Star Finder is a web app that helps you discover the most starred repositories on GitHub by programming language.

## Features

- 🔎 Search repositories by language
- ⭐ Find the most starred projects
- 📅 Filter by recent activity
- ♾️ Infinite scrolling

## Tech Stack

- Vite
- TypeScript
- GitHub API
- Vercel
